# proyecto_sprint_7
Proyecto Sprint 7 tripleten

Es una aplicación web que genera mediante un botón un histograma que muestra  la relación entre la el odómetro de los vehículos y la frecuencia en la que se presentan los valores de un dataframe sobre anuncios de autos usados en venta. 

también cuenta con un botón para generar un gráfico de dispersión que muestra la relación entre el odómetro y el precio
